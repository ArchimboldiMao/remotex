[远程] 易点互联招聘 软件需求分析员，软件架构师，软件开发，交互设计师，美工

6个月前

我们是贵州易点互联，一家专注软件开发实现的公司；

我们热衷并且只专注于技术实现（没错，一家全是猿人，攻城狮的公司）

世界那么大，我们在寻找我们的同类；即使我们不在一个地方，但我们的兴趣是一样的，即使相隔千山万水，我们也期望在网络上与你相知、相识并一起共事；一起讨论；一起进步

待遇

  1. 你的工作时间你自己安排，你可以一边游山玩水，田园耕种，周游世界，上着正常的班；一边抽时间做你敢兴趣的工作
  2. 按工作量评估费用、多劳多得、不劳不得
  3. 签订用工合同（明确双方权利、义务；报酬的计算、发放等）

流程

  1. 接受任务，评估工作量（费用）、交付时间点
  2. 我们每个流程都有验收流程和标准；把你的工作产出成我们需要的交付内容即可
  3. 根据你的产出进行验收并支付费用
  4. 任务基于我们的OA进行，你无须写任何工作报告

职位

软件需求分析员

  1. 负责分析客户的需求
  2. 必要的与客户的沟通（电话、QQ、微信、邮件等）
  3. 产出：《术语表》《软件需求说明书》《用例图》《业务流程图》

软件架构师

  1. 负责设计系统的架构
  2. 根据需求，确定使用的技术
  3. 根据需求，分解业务系统，分层功能模块，确定每个模块的边界
  4. 接口、技术定型
  5. 确定开发标准
  6. 产出：《架构设计文档》

交互设计师

  1. 根据需求、架构的设计，设计人机交互界面
  2. 需要完整表达出从主到次每个功能的交互细节
  3. 确定主要的功能交互结构，界面布局方式
  4. 可以大胆的创新交互方式
  5. 产出：《交互设计原型》

软件美工

  1. 根据《交互设计原型》，设计、美化界面效果
  2. 在不偏离《交互设计原型》的情况下，可以大胆的设计界面效果
  3. 切图
  4. 产出《界面原型图》

软件开发者

  1. 根据任务安排编码实现功能
  2. 任务安排中会说明：
    1. （输出）要达到的界面效果、功能目的 
    2. （输入）数据来源及其含义（数据库或者第三方）
    3. 其他注意事项
    4. 编码标准要求
  3. 产出：《源代码》《功能设计文档》（可选）


如果你在下列领域有兴趣，可发简历给我们（leeboo(at)yidianhulian.com），我们在寻找自由职业者；
