[北京/远程] 传太医医疗健康险招聘 PHP高手

6个月前

从去年第一次发招聘贴到现在，转眼已经一年了。公司的业务经历了很大的产品转型，从加号到现在依托医疗（大）数据的健康险。公司从初创期也走到了成长期，目前正在憋大招，需要各路高手的帮忙，无论是正式加入、远程顾问、兼职的方式我们都可以聊。

首先，描述一下公司的目前现状和需求：

  • 我们使用 PHP 和 Laravel ，基于阿里云的 ECS, RDS, OSS ，基础业务功能都已经实现，也使用融云整合了 IM 功能。
  • 后台和客户端分别使用 React.js 和 React Native 也开发了半年左右
  • 自动化方面实现了上线自动化，使用 Docker 搭建后端开发环境，一键 provision 本地环境等 (用了 Docker for Mac 真是好爽)。
  • 也使用工具来做项目管理、知识分享，每周都会有内部的技术分享。
  • 运维工程师很给力，我们的办公网络环境、会议室的音响（周五放电影），黑苹果都是我们自己搞定的。

公司到目前阶段，需要各种基础设施的升级，从代码级的重构到自动化单元测试、集成测试，以及需求管理的精细化，等等，最终要做到指哪打哪，开枪就爆头的程度~

  • 所以从技能上希望找到 PHP 、 Laravel 、 React.js 、 React Native 专精的小伙伴。
  • 从能力上希望找到喜欢主动解决问题的人，比如如何整合 laravel 和 react.js 代码到一个项目中，如何做到 npm private repository 和 docker registry 的多机同步，加速办公网络和开发、生产环境中的编译、部署。
  • 从合作方式上，目前我们有一位在加拿大远程工作的测试 leader ，所以远程的方式并不是问题。我们欢迎在二线城市，以及在美加生活比较安稳，需要实现点人生目标的同学以远程的方式加入。如果你在北京，最好的方式当然是在我们办公室里一起 happy 了，后面会放上办公室截图。

目前我们希望各路小牛能加入我们，公司也提供了成长和锻炼的空间，让你们从小牛升级到大牛~

现有的改进方向有：

  • Docker based APM, 为搭建 Micro Services 做准备，基于 Docker 1.12 ，实现开发和测试环境，以及 CI 的 Docker 化
  • Laravel 性能调优，包括各种异步、队列化
  • PHP code refactor, FP 化
  • React.js 和 React Native 的组件化
  • 业务流程的改进，数据结构设计
  • 项目管理成熟度的提升
  • 使用各种让人觉得 NB 的技术、工具改进工作效率

我们能提供的唯有机会和挑战，我们的业务在目前细分行业里跑的是最快的。在待遇方面，我们能提供业内中上的水平，对非一线城市和海外同胞来说是个不错的选择。对于以创业身份加入的人来说，我们也会给予最大的诚意。

有想法的小伙伴可以发送 email 到 yangliu@chuanty.com 来详聊。招人难，找到志同道合的人更难，我们不是在寻找一个雇员，而是在等待一起奋斗的战友。

薪水方面如果非要给一份范围的话，全职的是20k-35k左右
