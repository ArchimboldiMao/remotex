[北京/深圳/远程] BearyChat 招聘 iOS 开发工程师

6个月前

我们这次招聘主要面向 iOS 工程师，当然如果你是 前/后端、Android 开发/市场销售 等等岗位又对我们有点兴趣，也非常欢迎来和我们一起聊聊，我们期待所有志同道合的小伙伴加入我们的队伍，一起做这件在我们看来非常有意义的事。

照旧呢，还是要在最开始先介绍一下团队：

我们正在开发一款叫做 BearyChat 的团队沟通工具，希望能够帮助更多人借助分组讨论、第三方信息集成等的方式，提升工作效率，节约生命，愉快生活。

有兴趣了解更多可以查看我们的 官网

目前我们一共有 22 位同事，主要 base 在北京和深圳两处，也有一位同事在上海远程。所以如果你的坐标在北京或者深圳就最好不过了，当然，十分合适的话我们也不排斥远程。

我们现有的团队成员主要来自微软/阿里/网易/豆瓣之类，总之整个团队都比较年轻，互相之间可能比较容易有共同语言容易沟通。管理扁平、工程师文化之类的就不多说了。

我们对 iOS 工程师的期待

  • 有丰富的 OOP 经验；
  • 熟悉多线程，可以充分利用 run loop ；
  • 熟悉 MVVM 架构，熟练使用 RAC ；
  • 熟悉 UIKit 的实现机制；
  • 熟悉 Core Animation 、 Core Text 等 frameworks ；
  • 知道哪里有坑，哪里容易出现性能瓶颈；
  • 熟悉自动化测试；
  • 熟练使用 shell ；
  • 熟练使用各种 CI 工具；
  • 熟悉版本控制；
  • 严格遵守编码规范；
  • 英语流利；
  • 会用 Sketch ；
  • 能听懂设计师在说什么；
  • 熟悉各种协作工具。

「熟悉」指能够基于相关技术独自从头构建一个项目不留坑。

加分项

  • 经历过两个以上商业项目从零开始至线上稳定运营的过程；
  • 活跃于社区积极参与开源项目；
  • 有完善的知识管理系统并坚持持续学习。

关于福利待遇

  • 相对行业普遍水平有竞争力的薪酬，不会特别浮夸但是肯定要对得起你的能力；
  • 五险一金，年终奖，调薪之类就不多展开了；
  • 相对弹性的工作时间（弹性≠加班），双休；
  • 每月固定金额的饭补和房补；
  • 每月一天年假，可以攒着用；
  • 入职时按你的需要为你购买办公设备；
  • 有饮料零食之类不值得在招聘里提的东西，有不定期的 TB 和定期的团队旅行；
  • 其他内容你想到了可以提。
