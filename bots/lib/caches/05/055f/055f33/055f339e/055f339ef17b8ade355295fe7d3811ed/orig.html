[远程] 行业明星 SaaS 平台招专职 Web 前端和 Ruby

6个月前

随着互联网近 20 多年的发展，互联网已经在 C 端（消费端）发挥了巨大的影响力，几乎已经深入到人们日常生活工作的方方面面。但同时，我们认为，互联网为 C 端创造价值已经快到了极限， C 端流量为 BAT 、京东、小米这些公司所垄断，所剩下的创业创新机会已渺渺。

但从 B 端（企业端）看，互联网的机会还很多，这里几乎还是处女地。从另一个角度来说， B 端互联网将会是比 C 端互联网更为庞大的机会，其所能创造的价值甚至不夸张的说将会是 C 端互联网的 100 倍以上！

我们认为， SaaS 平台将会是 B 端互联网的核心战场。用 SaaS 去提升传统行业，将企业之间通过 SaaS 平台连接起来，在中国将会诞生几十家 BAT 级别的公司。

本着这样的认知，我们坚定的立足于 B 端 SaaS 领域创业。从逻辑上分析， SaaS 平台要创造商业价值，必然是那些离不开线下成交的行业，而二手车恰好就是这样一个行业，而且这还是一个价值巨大的、天然 O2O 性质的、正在迅猛飞速且还未被互联网“征服”的行业！

因此，我们从二手车 SaaS 开始！除了上述发展空间巨大的理由外，还包括如下两方面理由：

一方面，二手车销售成交离不开线下成交。同时，消费者为了一次性高效率线下看车洽谈，集中展示二手车的销售场所（主要是大型展厅和交易市场）必然仍将是未来二手车成交的主流模式。而这样集中展示二手车的线下销售场所，因为其经营的重资产特性，必然会使得品牌碎片化。这是二手车 SaaS 平台必然会出现的一个理由。

另一方面，随着各类互联网信息的越来越透明，以及各类汽车相关的产品和服务必然会越来越在线化（估值、认证、金融、保险、零配件、物流，等等等等），在二手车行业必然会大量使用各类移动 APP 应用涌现。大量通过移动 APP 形式展现的二手车相关应用，又是二手车 SaaS 平台整合资源、为二手车企业提供统一入口的另一个理由。

我们团队凭借着在这个行业的多年厮杀，已经积累了行业门槛和影响力，我们很有信心成为这个行业最大的、也真正平台性质的互联网公司。

我们团队的使命：让传统与时俱进！ 我们团队的价值观：技术创造价值！ 我们相信：望这一路漫长，相信就是力量！ 我们对人的期望：追求卓越，不甘平庸！敬畏传统、学习传统、再造传统！

现在我们需要扩建我们的组织，现迫切希望找到志同道合的真正工匠精神的程序员，与我们一起，在公司保证薪资略高于行业平均水平的基础上，通过远程工作模式，连接尽可能多的“技术情怀”的人，为我们的使命“让传统与时俱进”而努力！
