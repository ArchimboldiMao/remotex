[北京/远程] 墨刀 2016 前端工程师招聘(15-30k)

7个月前

想了很久我们和其他公司的区别， 貌似很多，又貌似没什么特殊的。 但有一点，我们可以骄傲并理直气壮的说，就是： 我们保证不加班。

创业也可以自由地拥有你的理想生活！

虽然墨刀是一家创业公司，但是我们可以保证不用加班、每周 8 x 5 工时、除法定节假日外的 10 天年假、五险一金以及年底双薪，并且，可以在家办公。

我们这次招聘是面向中国大陆所有地区，或许你觉得怀才不遇但又没有北漂的勇气，我们高效的远程协作管理方式，可以让你在你的家乡同样高效地与行业里的精英们一起战斗。

当然，如果你身在北京，也欢迎你来办公室和我们一起办公。我们只是为了避免京城的雾霾和刚涨价的交通费用让我们和一个怀着改变世界梦想的天才少年（或少女）擦身而过。

墨刀是什么？

墨刀是一个在线移动应用原型制作工具，旨在帮助产品经理，创业者快速制作可在手机端展示的移动应用原型。且可以远程团队协作。

团队情况：

团队目前有 5 个人， 3 个汉子、 2 个漂亮妹子（一个御姐一个软妹），我们不紧不慢的拿到了价格还不错的 A 轮投资，你要知道在这样的融资寒冬，能融到钱、价格还不错的，都是用硬实力说话的赚钱团队。

没错，我们的工具，已经有很多人愿意付费享受我们更实用的服务。

所以，一个从不过度承诺的我，这次终于敢说：我们的期权是很值钱的，并且我们希望团队成员在经过我们认可之后都有期权在手，肩并肩一起去改变世界。

我们提供：

  1. 月薪 15 – 30K
  2. 五险一金
  3. 10 天年假
  4. 年底双薪
  5. 每周 8×5 工时，不加班，周末可以自由参加活动
  6. MBPr/MBA + 外显
  7. 可在家办公（非北京）
  8. 读书 /社区分享奖励，每读一本书并在团队内部分享，或参加外部活动分享均有奖励

需要你具备：

  1. 熟练掌握 Javascript 及 Web API
  2. 熟练掌握 Linux/Git
  3. 熟悉 React
  4. 熟悉前端构建工具
  5. 具有单页 Web App 开发经验
  6. 良好的英语阅读及写作能力

最好还有（加分项）：

  1. 熟悉 Ruby on Rails
  2. 熟悉 React Native
  3. 熟悉 Electron
  4. 熟悉 iOS 、 Android 开发
  5. 热爱篮球或跑步
  6. 会打德州、桌游

我们的愿景：

虽然目前墨刀还只是一个 App 原型工具，但是我们希望在未来墨刀可以成为设计团队的 Github ，不止是工具，更是一个在线协作平台与开放式社区，期待你加入我们一起实现这个目标。

我们的办公地标：

三元桥时间国际（离地铁三元桥站很近，周围美食很多，你懂的）。
